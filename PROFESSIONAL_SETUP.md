## 🔧 **Setup Instructions:**

### 1. Install Dependencies
```bash
# Backend
cd backend
npm install

# Frontend
cd ../frontend
npm install
```

### 2. Start Services
```bash
# Terminal 1 - Backend
cd backend
npm start

# Terminal 2 - Frontend
cd frontend
npm start
```

### 3. Access Application
- **Landing Page:** http://localhost:3001
- **User Login:** http://localhost:3001/login
- **Admin Login:** http://localhost:3001/login
